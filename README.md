# Stanford-algorithm
This repo contains my solutions to the four stanford algorithm courses assignments, coded in c++11 and python 3.0
## Usage
### Cpp: 
Use a compiler support c++11 or higher, compile command:
```
g++ solution.cpp -std=c++11 -Wall
```

Run the program:
```
./a.out input.txt
```

### Python: 
Run program command:
```
python3 solution.py
```
